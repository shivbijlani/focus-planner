# Future Capabilities

Unordered backlog of *what* the user could do. Out of scope for V1. UI-agnostic.

## Storage & sync
- Work on non-Chromium browsers (iOS Safari, Firefox) via an alternative storage backend.
- Sync a folder with OneDrive / Google Drive / Dropbox directly from the app (not via the OS sync client).
- Paid encrypted cross-device sync without relying on a third-party drive.
- See and resolve conflicts when the same file was edited on two devices.
- Get live updates when an external process edits the folder.

## Task abilities
- Search across all tasks and journals.
- Tag a task with arbitrary labels (beyond priority and manager priority).
- Make a task recurring (daily, weekly, custom).
- Give a task a due date and see it highlighted when overdue.
- Receive a reminder for a task at a chosen time.
- Track time spent on a task.
- Run a pomodoro session tied to a task.

## Journal abilities
- Paste an image into a journal and have it saved alongside the markdown.
- Record a voice note and have it transcribed into a bullet.
- Keep a daily journal separate from per-task journals.

## Intelligence (opt-in, user-supplied keys)
- Enter a task in natural language ("remind me to call mom tomorrow") and have fields filled in.
- Get a suggested priority from task text.
- Get an auto-generated weekly review from the completed archive.
- See suggested groupings of related tasks.

## Collaboration (paid, much later)
- Share a folder with another person via invite link.
- Assign a task to a specific person.
- See who else is viewing a file right now.
- Keep everything markdown and file-first even when collaborating.

## Import
- Import tasks from Todoist, TickTick, Things, Obsidian Tasks.

## Monetization shape (informs architecture, not built)
- Free: local folder, single device, unlimited tasks.
- Paid 1: encrypted cloud sync.
- Paid 2: sharing and teams.
- Paid 3: hosted AI assist.
